## Getting Start

开始使用

<br />

### Install

Using `npm` to install

```bash
npm install @pekings/r-ui-component --save
```

Using `yarn` to install

```bash
yarn add @pekings/r-ui-component
```

### Import

```js
// 全局引入样式
import "@pekings/r-ui-component/dist/index.css";
// 引入组件
import { Button } from "@pekings/r-ui-component";
```
